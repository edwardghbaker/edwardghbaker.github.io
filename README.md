# Personal Portfolio — quick start

This repository contains a minimal static portfolio and CV for Edward G. Baker.

Files added:
- [index.html](index.html) — main landing page
- [styles.css](styles.css) — basic styling
- [cv.html](cv.html) — editable HTML CV
- [cv.pdf](cv.pdf) — placeholder PDF (replace with your real resume)

Preview locally:

PowerShell / Windows CMD:

```powershell
# from the repo root
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Customize:
- Update the contact details in `cv.html` and `index.html`.
- Replace `cv.pdf` with your real PDF to enable the download button.
- Add links and project descriptions under the Projects section.
